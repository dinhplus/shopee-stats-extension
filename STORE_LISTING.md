## Detailed Description
📊 **Shopee Thống Kê Chi Tiêu** - Extension miễn phí giúp bạn xem tổng quan chi tiêu trên Shopee.vn

### 🎯 Tính Năng Chính:
✅ **Thống kê tổng quan**
   - Tổng chi tiêu tất cả thời gian
   - Tổng số đơn hàng
   - Tổng số sản phẩm đã mua
   - Tổng tiền tiết kiệm từ khuyến mãi

✅ **Thống kê theo thời gian**
   - 1 tháng gần nhất
   - 3 tháng gần nhất
   - 6 tháng gần nhất
   - 1 năm gần nhất

✅ **Thống kê theo năm**
   - Chi tiêu từng năm
   - So sánh giữa các năm

✅ **Lưu trữ dữ liệu**
   - Dữ liệu được lưu cục bộ
   - Xem lại mọi lúc không cần chạy lại

### 🔒 Bảo Mật & Quyền Riêng Tư:
- ✅ **100% dữ liệu local** - Không gửi dữ liệu đi đâu
- ✅ **Open source** - Code công khai trên GitHub
- ✅ **Không quảng cáo** - Không tracking
- ✅ **Hoàn toàn miễn phí**

### 📱 Cách Sử dụng:
1. Đăng nhập Shopee.vn
2. Click vào icon extension
3. Nhấn "Bắt Đầu Thống Kê"
4. Chờ extension thu thập dữ liệu
5. Xem kết quả ngay trên popup!

### 🛠️ Công Nghệ:
- TypeScript + React
- Chrome Extension Manifest V3
- Vite Build System

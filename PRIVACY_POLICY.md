# Privacy Policy for Shopee Thống Kê Chi Tiêu

**Last updated: February 4, 2026**

## Data Collection
This extension **does not collect, store, or transmit any personal data** to external servers.

## Data Processing
- All order statistics are processed **locally** on your device
- Data is only read from Shopee.vn when you explicitly click "Bắt Đầu Thống Kê"
- Statistics are stored locally using Chrome's storage API for persistence

## Permissions Used
- **activeTab**: To inject scripts into Shopee.vn tabs
- **scripting**: To run statistics collection on Shopee pages
- **storage**: To save statistics locally for offline viewing
- **host_permissions (shopee.vn)**: To access order data from Shopee API

## Data Storage
- Statistics are stored **only in your browser** using `chrome.storage.local`
- No data is sent to any third-party servers
- You can clear all data by resetting the extension or uninstalling it

## Third-Party Services
This extension does not use any third-party services or analytics.

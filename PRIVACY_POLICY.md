# Privacy Policy for Shopee Thống Kê Chi Tiêu

**Last updated: February 27, 2026**

## Overview
This extension is an unofficial tool not affiliated with, endorsed by, or sponsored by Shopee. It helps users view their own spending statistics on Shopee.vn.

## Data Collection
This extension **does not collect, store, or transmit any personal data** to external servers.

## Data Processing
- All order statistics are processed **locally** on your device
- Data is only read from Shopee.vn when you explicitly click "Bắt Đầu Thống Kê"
- Statistics are stored locally using Chrome's storage API for persistence
- No data is shared with the extension developer or any third parties

## Permissions Used
- **activeTab**: To inject scripts into Shopee.vn tabs only when you click the extension
- **scripting**: To run statistics collection on Shopee pages
- **storage**: To save statistics locally for offline viewing
- **host_permissions (shopee.vn)**: To access your order data from the Shopee API on your behalf

## Data Storage
- Statistics are stored **only in your browser** using `chrome.storage.local`
- No data is sent to any external or third-party servers
- You can clear all data by clicking "Bắt Đầu Lại" in the extension or by uninstalling it

## Third-Party Services
This extension does not use any third-party services, analytics, or tracking tools.

## User Data Handling
- **Data collected**: Order history data (order totals, item counts, dates) from your Shopee account
- **How it's used**: To calculate and display spending statistics locally
- **Data retention**: Stored locally until you reset or uninstall the extension
- **Data sharing**: None - no data leaves your browser
